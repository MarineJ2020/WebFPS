Drop level JSON files in this folder to make them appear in the main menu level selector.

Accepted shapes:
- A map manifest entry with `id`, `displayName`, and `definition`.
- A level editor export with `version: 1`, `id`, `name`, `volumes`, `spawnPoints`, and `navMeshRegions`.
- A raw runtime `MapDefinition` with `volumes`, `spawnPoints`, and `navMeshRegions`.
